## 📖 Overview

**Motion Lab** is a high-end interaction laboratory built with React, GSAP, and Framer Motion. It serves as both a visual showcase for cutting-edge UI/UX patterns and a **Developer Toolkit** where users can explore, customize, and copy production-ready code snippets.

### Core Philosophy

* **Performance First**: 60fps animations driven by GPU-accelerated transforms.
* **Developer Empowerment**: Integrated DevTools for instant code portability.
* **Motion Theater**: Combining scroll-choreography with 3D perspective and glassmorphism.

---

## ✨ Features

### 🚀 The Experience

* **Galactic Hero**: Canvas-based particle galaxy system with mouse-repulsion.
* **Smooth Navigation**: Lenis-driven physics scroll for zero-jank movement.
* **Magnetic Cursor**: Lerp-based ring tracker with magnetic attraction to interactive elements.
* **Global Theme Engine**: Real-time accent color shifting and light/dark mode persistence via Zustand.

### 💻 Snippets Hub (DevTools)

The **DevTools** section is a central hub for all interaction code used in the project.

* **50+ Snippets**: Ready-to-copy JS Hooks, CSS utilities, and React components.
* **Categories**: CSS Glassmorphism, Magnetic Hooks, Tilt Effects, Ripple Utils, etc.
* **Syntax Highlighting**: Real-time code preview with a custom dark-mode tokenizer.
* **Copy-Ready**: One-click clipboard functionality for individual variants.

## 📦 Installation

### Requirements
* Node.js 18+
* npm / yarn

### Setup
```bash
# Clone the repository
git clone https://github.com/d-ghosh717/motion-lab.git

# Enter directory
cd motion-lab

# Install dependencies
npm install

# Run dev server
npm run dev
```

### Build
```bash
npm run build
npm run preview
```

## ⚡ Performance Monitoring

The project includes a built-in **Performance Panel** to monitor the impact of animations in real-time:
* **Live FPS Counter**: Real-time rendering speed.
* **Memory Gauge**: Visualization of active animation loops.
* **Effects Toggle**: Disable complex particles or blurs for low-end device testing.
